Full-stack CRM system for managing leads, sales funnels, and client activity.

## Features
- Multi-tenant architecture (data isolation per client)
- Authentication (Supabase Auth)
- Lead management (search, filters, pagination)
- Sales funnel tracking
- Activity logging

## Tech Stack
- Next.js (React)
- TypeScript
- Supabase (PostgreSQL, Auth, RLS)
- Tailwind CSS
